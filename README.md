# C Image Manipulation
Just testing image manipulation with C *(no external libraries)*

# How to run
## Install the requirements
```sh
sudo apt install make clang
```

## Running
```sh
make
./colorize input.bmp output.bmp
# Or run the shell file: run.sh
```
